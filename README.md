# oci-instance
The repo for deploying and managing the OCI instance responsible for hosting my personal website @mrlouf.com

## Setup

After using all my credits on the AWS free tier account that I had, I explored other options for cloud providers and I found out that Oracle Cloud Infrastructure (OCI) offers an infinite free tier.

The available resources in the free tier are, of course, limited, but they are more than enough for hosting a personal website.
